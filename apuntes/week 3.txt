* CRUD Operations with the Node.js Driver
    - Read => db.collection(collectionname)
        + .find({}) => Returns a Cursor object, that will retreive data in batches. It is synchronus
        + .find({}).toArray((err, docs)) => Returns an Array object, once it fetch all the results of the dataset. It is asynchronus
        + cursor.project(projection) => Return only the fields specified on the projection specified as parameter
            - projection => { field: visible }
        + cursor.sort(sort)
            - sort => { field: order}
                + Order => Ascending or descending (1 or -1)
            - sort => [[field1: order], [field2: order]]
        + cursor.skip(number)
            - Number => Amount of documents to be skipped when getting the results back
        + cursor.limit(number)
            - Number => Limit amount of documents being retreive from the database
    - Create => db.collection(collectionname)
        + .insertOne({ object })
            - Returns {'acknowledged': '', 'insertedId': ''}
                + Acknowledged => Command succeded (true - false)
                + InsertedId => Unique identifier, generated by MongoDB if we do not provide _id on the object passed as parameter
        + .insertMany([{ object1 }, { object2 }])
             - Returns {'acknowledged': '', 'insertedIds': '[]'}
                + Acknowledged => Command succeded (true - false)
                + InsertedIds => Unique identifier, generated by MongoDB if we do not provide _id for all the objects passed as parameter
    - Delete => db.collection(collectionname)
        + .deleteOne({ filter })
            - Returns {'acknowledged': '', 'matchedCount': ''}
                + Acknowledged => Command succeded (true - false)
                + MatchedCount => Quantity of matched documents
        + .deleteMany()    
            - Returns {'acknowledged': '', 'matchedCount': ''}
                + Acknowledged => Command succeded (true - false)
                + MatchedCount => Quantity of matched documents
